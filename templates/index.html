MSU Grades API

This API was build using data gathered from MSU Grades. It aims to make the data from MSU Grades easily accessible for other applications.


Documentation

In order to use this API, an application needs to query "https://www.msugradesapi.com/grades/". At the end of the URL, add the course code and term separated by underscores '_'. Follow the example below, or go to the "Try it out" section of this site to build a URL for an API query. Also included in the example below and in the "Try it out" section is how the results will be returned from an API call.

Example request url to API:

https://www.msugradesapi.com/grades/CSE_325
